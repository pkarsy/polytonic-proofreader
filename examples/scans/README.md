# Example scan images

These images are **heavily downscaled** (800 px wide, ~150–200 KB each) and are provided **only as a quick way to test the editor**.

They are cropped from **ΤΟ ΣΥΜΠΑΝ (THE UNIVERSE)** — a Greek-language encyclopedia printed in Smyrna in 1888.

- 📖 **Full scanned book (high resolution):** [archive.org/details/to-sympan-1888/](https://archive.org/details/to-sympan-1888/)
- 📂 **OCR project (full scans + text):** [github.com/pkarsy/to-sympan-1888](https://github.com/pkarsy/to-sympan-1888)
